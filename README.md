# ckit-files

This repository contains configuration files that can be used to get you started with [ckit](https://fpgmaas.github.io/ckit/). In order to use these configuration files, make sure you have ckit installed and simply place them in the `~/ckit` directory.

Alternatively, run the following command to automatically download the files from the `config-files` into your `ckit` home directory:

```shell
ckit init --skip-local --download-global-defaults
```



For more information, see the [documentation](https://fpgmaas.github.io/ckit/) of ckit.
